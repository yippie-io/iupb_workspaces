Getting started

Here’s how to get rolling:

  • Find a building

    To see all available buildings, GET /buildings.json.

  • Find workplaces in a building

    To see all available workplaces of a building, GET /buildings/:FACULTY_ID/workplaces.json.

    For your convenience: You also find the workplaces in the response for faculties if you add the include=workplaces parameter.

  • Get all workplaces or filter for properties

    ...., GET /....json.

JSONP

For any url, use the .jsonp extension followed by your callback, like in:
/buildings.jsonp?callback=jsonpsuccess
